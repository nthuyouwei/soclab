the common instruction of git
```
git clone <URL>
git add <file> // . 代表全部
git commit -m "input_anything"
git push
git status
git log
git config --global --list
```

```

cp // 拷貝
pwd // 列印工作路徑
ls //  目錄清單
mkdir // 建立目錄
touch //建立文件
man // 使用說明
vim //編輯畫面
```
